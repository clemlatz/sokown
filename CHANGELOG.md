# Changelog

All notable features added to Sokown from the player's perspective.

## 2026-01-18

- You can now toggle a visual trajectory line on the star map showing your ship's planned path to its destination
- The trajectory line features a smooth animation effect for better visualization

## 2026-01-16

- Receive email notifications when your ships arrive at their destinations

## 2026-01-15

- Hover over any location on the star map to see its precise coordinates

## 2024-05-17

- The star map now remembers your zoom level preference between sessions

## 2024-05-16

- Improved default zoom level for a better initial view of the solar system

## 2024-05-14

- Location coordinates now display with cleaner two-decimal precision

## 2024-05-13

- Use zoom controls to navigate the star map more easily
- Ships now maintain their heading when arriving at destinations

## 2024-05-12

- Each ship now has its own star map showing its current position in the solar system
- See all ships displayed on the star map with their real-time positions
- The star map automatically centers on your current ship
- Star map now uses actual astronomical coordinates for accurate visualization

## 2024-05-09

- View your ship's current heading/course on the ship detail page
- Each celestial body now displays with its own distinctive color
- Added visual lighting effects on planets for enhanced appearance
- Very small celestial bodies are automatically hidden to reduce clutter
- Ships automatically update their course as they travel through space

## 2024-05-08

- New visual star map showing the solar system and celestial bodies
- See the distance from each location to the Sun

## 2024-05-05

- Calculate and view where celestial bodies will be at any given time

## 2024-05-03

- Mercury and Venus are now available as destinations

## 2024-05-02

- Visit individual detail pages for each celestial location
- Location positions now shown with three decimal digits

## 2024-04-26

- See arrival times in friendly relative format (e.g., "in 2 hours")

## 2024-04-23

- View the estimated time of arrival (ETA) for traveling ships
- Autopilot form now prefills with your ship's current settings
- Autopilot fields are now required to prevent navigation errors
- Autopilot fields show empty by default when not actively traveling

## 2024-04-22

- See how long it will take to reach your destination
- All pages now have proper titles for easier navigation
- Game now uses real kilometers for all measurements
- Position coordinates now use monospace font for better readability

## 2024-04-21

- Ships now track and display their current location
- Your ship's location updates automatically when you arrive at destinations

## 2024-04-15

- Ships now move with the orbital motion of their current location

## 2024-04-14

- All celestial bodies now move based on real astronomical calculations
- Browse all locations in the solar system from a dedicated page
- Celestial body positions update every minute to reflect actual orbital mechanics
- Ship names must be unique - you'll see an error if you try to use a taken name

## 2024-04-13

- Pilot names must be unique - you'll see an error if you try to use a taken name

## 2024-04-12

- "Log in" wording is now consistent throughout the application
