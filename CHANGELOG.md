# Changelog

All notable features added to Sokown from the player's perspective.

## 2026-01-18

### Added
- **Trajectory Lines**: You can now toggle a visual trajectory line on the star map showing your ship's planned path to its destination
- **Trajectory Animation**: The trajectory line features a smooth animation effect for better visualization

## 2026-01-16

### Added
- **Email Notifications**: Receive email notifications when your ships arrive at their destinations

## 2026-01-15

### Added
- **Coordinate Display**: Hover over any location on the star map to see its precise coordinates

## 2024-05-17

### Added
- **Persistent Zoom**: The star map now remembers your zoom level preference between sessions

## 2024-05-16

### Updated
- **Star Map View**: Improved default zoom level for a better initial view of the solar system

## 2024-05-14

### Updated
- **Coordinate Precision**: Location coordinates now display with cleaner two-decimal precision

## 2024-05-13

### Added
- **Star Map Zoom**: Use zoom controls to navigate the star map more easily

### Updated
- **Ship Navigation**: Ships now maintain their heading when arriving at destinations

## 2024-05-12

### Added
- **Star Map on Ship Pages**: Each ship now has its own star map showing its current position in the solar system
- **Ship Visualization**: See all ships displayed on the star map with their real-time positions
- **Auto-Centering**: The star map automatically centers on your current ship
- **Real Coordinates**: Star map now uses actual astronomical coordinates for accurate visualization

## 2024-05-09

### Added
- **Ship Course Display**: View your ship's current heading/course on the ship detail page
- **Colored Planets**: Each celestial body now displays with its own distinctive color
- **Planet Lighting**: Added visual lighting effects on planets for enhanced appearance
- **Smart Display**: Very small celestial bodies are automatically hidden to reduce clutter

### Updated
- **Course Tracking**: Ships automatically update their course as they travel through space

## 2024-05-08

### Added
- **Interactive Star Map**: New visual star map showing the solar system and celestial bodies
- **Distance Information**: See the distance from each location to the Sun

## 2024-05-05

### Added
- **Future Positions**: Calculate and view where celestial bodies will be at any given time

## 2024-05-03

### Added
- **New Locations**: Mercury and Venus are now available as destinations

## 2024-05-02

### Added
- **Location Pages**: Visit individual detail pages for each celestial location

## 2024-04-26

### Added

## 2024-04-23

### Added

### Added
- **Location Tracking**: Ships now track and display their current location
- **Automatic Updates**: Your ship's location updates automatically when you arrive at destinations

## 2024-04-15

### Added

## 2024-04-14

### Added
- **Real Astronomy**: All celestial bodies now move based on real astronomical calculations
- **Locations Page**: Browse all locations in the solar system from a dedicated page

## Feature Highlights

### Star Map & Visualization
- Interactive star map with zoom controls and persistent settings
- Real-time ship and location positions using actual astronomical data
- Animated trajectory lines showing your ship's path
- Color-coded planets with lighting effects
- Hover to view coordinates
- Auto-centering on your ship

### Navigation & Travel
- Real-time course/heading tracking
- Time to destination and ETA displays
- Relative arrival times (e.g., "in 2 hours")
- Smart autopilot with prefilled values
- Ships move with orbital mechanics

### Locations & Solar System
- Real astronomical calculations updated every minute
- Mercury, Venus, Earth, and other celestial bodies
- Individual pages for each location
- Distance from the Sun displayed
- Future position predictions

### Player Experience
- Email notifications when ships arrive
- Unique pilot and ship names
- Clean coordinate displays
- Real-world scale using kilometers
